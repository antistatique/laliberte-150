```

```

### Working locally

The theme is a submodule
```
$ git submodule init
$ git submodule update
```

#### warning
When working on the theme, you must first commit theme changes then commit new commit ref to the submodule

```
$ cd themes/liberte
$ git add, commit, push
$ cd ../..
$ git add, commit, push
```
